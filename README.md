# homography-assistant


## Requirements

- Node.js
- yarn
  ```bash
  npm install -g yarn
  ```
- Python ( `>3.6` )

## Set-Up

### Frontend
```bash
cd frontend
yarn 
```

### Backend
```bash
cd Backend
pip3 install -r requirements.txt
```

## Run App

1. Exec following `Frontend` and `Backend`
2. Open `http://localhost:8080` in browser

### Frontend
```bash
cd frontend
yarn serve
```

### Backend
```bash
cd backend
python3 app.py
```
